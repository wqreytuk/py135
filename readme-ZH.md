用法：
```
入域主机
python 1.py a.b/Administrator 12345 192.168.1.1

python 1.py a.b/Administrator :nthash 192.168.1.1

工作组

python 1.py ./Administrator 12345 192.168.1.1

python 1.py ./Administrator :nthash 192.168.1.1


最后有一个可选项，是编码方式

文件上传命令格式：
	up$本地文件路径$远程文件路径
	
文件下载命令格式：
	down$远程文件路径$本地文件路径
  
命令执行：
	直接输入命令即可
```
